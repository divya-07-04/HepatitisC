# Hepatitis C Disease Prediction System

A Machine Learning based web application developed using Flask to predict Hepatitis C disease from user input data.

## Project Overview
This project helps in predicting the possibility of Hepatitis C disease using trained Machine Learning models. The application provides a simple and interactive web interface for users to enter medical parameters and receive prediction results.

## Features
- Hepatitis C disease prediction
- Flask-based web application
- Interactive user interface
- Fast and simple prediction system
- Machine Learning integration

## Technologies Used
- Python
- Flask
- HTML
- CSS
- Machine Learning
